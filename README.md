# МастерОк — лендинг платформы для поиска специалистов

## Тема проекта
Платформа для поиска специалистов с прозрачными условиями и безопасной сделкой.

## Структура лендинга
- Хедер с навигацией
- Hero-секция с поиском
- Категории услуг
- О проекте
- Как это работает (3 шага)
- Отзывы клиентов
- CTA-секция
- Футер с контактами

## Адаптивность
- Desktop: 3 колонки
- Tablet: 2 колонки
- Mobile: 1 колонка
